# Weather App 

This is a app to check weather. You make account, login, then check weather for any city. Very easy!

# working

- create account with username and password
- Login to your account  
- Check weather for any city

# to use

# step 1 - Download (clone ) git repo


# step 2 -create venv and install requirements using pip install


# step 3 - go to openweathermap and get api key

# step 4 - paste api key in .env file and import api key from .env to main file

# step 5 - Run

# output

- Type `2` for make new account
- Type `1` for login and see weather
- Type `3` for exit

## files inside

- `login.py` - The main program
- `.env` - Your secret API key (don't upload to github!)
- `users.json` - Your accounts data
- `.gitignore` - Tell git what not upload
